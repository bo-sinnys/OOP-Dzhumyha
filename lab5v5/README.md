# Лабораторна робота №5: Рейтинги фільмів

### Опис задачі
Ця програма дозволяє додавати рейтинги до фільмів та обчислювати середній рейтинг, враховуючи певні обмеження.

### Запуск
1. Скомпілюйте і запустіть проект.
2. Програма додасть 5 рейтингів до фільму "Inception" і обчислить середній рейтинг (з відсіканням мінімального та максимального значень).
